### 1.3 Exercises

a. If $a$ and $b$ are integers and $a+b=a$, then $b=0$.

| Step    | Justification          |
| ------- | ---------------------- |
| $a+0=a$ | Identity               |
| $0+a=a$ | Commutativity          |
| $b+a=a$ | Substitution of Equals |
| $a+b=a$ | Commutativity          |

b. If $a$, $b$, $c$ and $d$ are integers, then $(a+b)+(c+d)=(a+c)+(b+d)$.

| Step          | Justification |
| ------------- | ------------- |
| $a+(b+c)+d$   | Associativity |
| $a+(c+b)+d$   | Commutativity |
| $(a+b)+(c+d)$ | Associativity |
